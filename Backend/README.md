# AI Caption Generator FastAPI Application

A FastAPI service that generates creative captions using AI models from Groq and RunPod.

## Features

- Generate random funny captions using Groq AI
- Generate content from custom prompts using RunPod AI
- RESTful API with proper error handling
- Health check endpoints for service monitoring
- Comprehensive documentation with Swagger UI
